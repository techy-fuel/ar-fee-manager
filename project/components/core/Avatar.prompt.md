Initials avatar for students, parents and staff. Color is derived deterministically from the name.

```jsx
<Avatar name="Ahmed Raza" size="md" />
<Avatar name="Fatima Bibi" src="/photo.jpg" size="lg" />
```

Sizes `xs`–`xl`. Pass `square` for a rounded-square (used in receipt headers / settings). Always pass `name` so initials and tint resolve.
