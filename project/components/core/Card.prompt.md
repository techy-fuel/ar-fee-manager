The base surface for every panel — chart cards, tables, forms, profile blocks.

```jsx
<Card title="Monthly Collection" subtitle="July 2025" actions={<Button size="sm" variant="ghost">Export</Button>}>
  …chart…
</Card>
```

Use `padding="none"` when wrapping a full-bleed table. Set `interactive` for clickable cards (student rows expanded into cards). `footer` renders on a sunken bar — good for totals and pagination.
