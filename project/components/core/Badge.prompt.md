Compact status label — use for student status (Active/Inactive), fee status (Paid/Pending/Overdue), and counts.

```jsx
<Badge variant="success" dot>Paid</Badge>
<Badge variant="warning" dot>Pending</Badge>
<Badge variant="danger" dot>Overdue</Badge>
<Badge variant="neutral">Class 7-B</Badge>
```

Variants map to status colors: `success` (paid/active/collected), `warning` (pending), `danger` (overdue/inactive), `info`/`brand` (informational), `neutral` (categories). Add `dot` for a leading status indicator.
